1. Se citeste un numar intreg. Sa se afiseze urmatorul numar.

Exemple:

  • se citeste 3, se afiseaza 4
  • se citeste -4, se afiseaza -3

2. Se citeste un un numar intreg. Sa se afiseze opusul lui

Exemple:

  • se citeste 3, se afiseaza -3
  • se citeste -4, se afiseaza 4
  • se citeste 0, se afiseaza 0

3. Se citeste un un numar intreg. Sa se afiseze daca este pozitiv, negativ sau nul.

Exemple:

  • se citeste 3, se afiseaza “S-a introdus o valoare pozitiva”
  • se citeste -4, se afiseaza “S-a introdus o valoare negativa”
  • se citeste 0, se afiseaza “S-a introdus o valoare nula”

4. Se citesc doua numere. Sa se interschimbe valorile in cele 2 variabile.

Exemple:

  • se citesc 3 si 2, se afiseaza 2 3
  • se citesc -3 si 2, se afiseaza 2 -3

5. Se citeste un numar intreg pozitiv. Sa se afiseze toate numerele pozitive nenule mai mici sau egale decat valoarea citita, in ordine crescatoare

Exemple:

  • Se citeste 5. Se afiseaza 1 2 3 4 5

6.Se citeste un numar. Sa se afiseze toti divizorii numarului citit

Exemple:

  • Se citeste 12. Se afiseaza 1 2 3 4 6 12
  • Se citeste 11. Se afiseaza 1 11
  • Se citeste 9. Se afiseaza 1 3 9

7.Se citeste un sir de caractere. Sa se afiseze procentajul de vocale. Se ignora spatiile albe in calculul caracterelor.

Exemple:

  • Se citeste “Ana are mere”. Se afiseaza: 60%
