Scurtaturi VS Code pentru HTML CSS JS

! + enter - Creeaza sablonul de HTML

Numele tagului, exemple h1, img, a + enter Creeaza un tag gol cu numele dat

CTRL + S - Save

CTRL + C - copiere text selectat sau linie curenta

CTRL + V - lipire/paste

CTRL + X - Cut/decupare text selectat sau linie curenta

CTRL + Y - redo

CTRL + Z - undo

CTRL + A - Select all

ALT + ↓ / ↑ - Mutare linie curenta sus sau jos

ALT + SHIFT + ↓ / ↑ - Duplica linia curenta sus sau jos
